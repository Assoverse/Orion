What is Orion?

Orion brings Kubernetes-style declarative orchestration to JavaScript without the complexity of containers. Perfect for development, testing, and lightweight production deployments.

üéØ

Declarative Configuration

Define your services, replicas, and ports in YAML. Orion handles the rest with reconciliation loops.

üì°

Real-time Control Plane

TypeScript-powered Fastify server with WebSocket reconciliation. REST + WebSocket API for full control.

Auto-restart & Heartbeats

Agents supervise processes with automatic restarts on failure. Health checks keep services alive.

Next.js Dashboard

Core Features

01

TypeScript Control Plane

Full TypeScript control plane powered by Fastify with WebSocket reconciliation loops. REST + WebSocket API for complete cluster management. Round-robin scheduler distributes services across nodes intelligently.

  • Fastify-based REST API
  • WebSocket reconciliation loops
  • Round-robin scheduler
  • Zod-based schema validation
02

Lightweight Agents

Node 20+ agents run on each machine, supervising processes with restart policies. Stream metrics, heartbeats, and logs to the control plane. No container overhead.

  • Process supervision with restart policies
  • Real-time metrics streaming
  • Heartbeat health checks
  • Minimal resource footprint
03

Declarative CLI

Kubernetes-inspired CLI with commands like apply, get, logs, and exec. Commander-based with Chalk/Ora UX for beautiful terminal output.

  • gorion init - Initialize configuration
  • gorion apply - Deploy services
  • gorion get - View resources
  • gorion logs - Stream logs
04

Next.js Dashboard

Production-ready dashboard built with Next.js 14 and Tailwind CSS. Real-time service status, node metrics, and live log streaming. Beautiful, responsive, and fast.

  • Service and node overview
  • Real-time metrics charts
  • Live log streaming
  • Responsive Tailwind UI
05

DNS Manager

Integrated multi-provider DNS automation supporting DigitalOcean, Cloudflare, AWS Route53, wildcard proxy, and local providers. Optional wildcard SSL automation with Let's Encrypt.

  • Multi-cloud DNS providers
  • Wildcard SSL automation
  • Service discovery integration
  • Automatic record updates

Quick Start

1

Install Dependencies

pnpm install
2

Launch Full Stack

scripts/dev.sh

Starts control plane, agent, and dashboard

3

Deploy Configuration

pnpm --filter @orion/cli run dev
gorion init
gorion apply -f orion.yaml
4

Open Dashboard

Visit http://localhost:3000

Example Configuration

apiVersion: orion/v1
kind: Config
metadata:
  name: demo
  environment: dev
spec:
  services:
    - name: api-users
      path: ./examples/simple-service
      entry: src/index.ts
      runtime: node
      replicas: 2
      env:
        NODE_ENV: development
      ports:
        - name: http
          port: 4000

    - name: web
      path: ./examples/next-app
      type: nextjs
      runtime: node
      replicas: 1

  nodes: []
