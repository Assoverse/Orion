Usage

Deploy Your First Service

1. Initialize CLI

pnpm --filter @orion/cli run dev
gorion init

2. Create Configuration

apiVersion: orion/v1
kind: Config
metadata:
  name: demo
  environment: dev
spec:
  services:
    - name: api-users
      path: ./examples/simple-service
      entry: src/index.ts
      runtime: node
      replicas: 2
      env:
        NODE_ENV: development
      ports:
        - name: http
          port: 4000

    - name: web
      path: ./examples/next-app
      type: nextjs
      runtime: node
      replicas: 1

3. Deploy

gorion apply -f orion.yaml

4. View Services

gorion get services

# Output:
NAME        STATUS    REPLICAS   NODES
api-users   Running   2/2        node-01
web         Running   1/1        node-02

5. Open Dashboard

Visit http://localhost:3000 for the visual dashboard

CLI Commands

Command Description
gorion init Initialize configuration
gorion apply -f <file> Deploy services from config file
gorion get services List all services
gorion get nodes List all nodes
gorion logs <service> Stream service logs
gorion exec <service> Execute command in service
